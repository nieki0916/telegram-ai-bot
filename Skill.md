# 社情民意写作（九三学社版）

用于撰写规范的社情民意报告，供九三学社社员、政协委员、人大代表等向政府反映问题、提出建议。

## 使用流程

### 1. 获取标题

询问用户提供社情民意的标题。标题应简洁明了，点明主题。

### 2. 资料收集

**优先询问用户：**
- 是否有本地文件/资料需要参考？如有，请用户提供文件路径
- 是否有特定的背景资料或数据需要纳入？

**然后执行网络搜索：**
- 使用 `web_search` 搜索与标题相关的政策、新闻、数据
- 搜索关键词建议：标题主题 + "问题"、"现状"、"政策"、"建议"
- 如有需要，使用 `web_fetch` 获取详细内容

### 3. 分析标题

根据收集的资料，分析标题涉及的核心问题：
- 当前政策背景是什么？
- 涉及哪些利益相关方？
- 问题的紧迫性和重要性如何？

### 4. 撰写报告

报告总字数约 **1500字**，分为三个部分：

#### 第一部分：概述（200-300字）

描述当前情况的总体面貌：
- 政策背景（国家/地方相关政策）
- 现状描述（目前发展到什么阶段）
- 重要性说明（为什么值得关注）

写作要点：客观陈述，数据支撑，条理清晰。

#### 第二部分：提出问题（约500-600字）

陈述存在的主要问题，可分2-3个小点：
- 问题一：具体表现、影响范围、典型案例
- 问题二：深层原因、关联问题
- （可选）问题三：衍生影响

写作要点：
- 问题要具体，避免空泛
- 有数据或案例支撑
- 分析问题产生的原因

#### 第三部分：提出建议（约600-700字）

针对问题提出3-5条建议：
- 建议要**合理**（符合法律法规和政策方向）
- 建议要**有针对性**（直接回应前文提出的问题）
- 建议要**有可操作性**（有明确的责任主体和实施路径）

每条建议建议包含：
- 建议内容
- 理由依据
- 实施主体（哪个部门负责）
- 具体措施（如何落实）

### 5. 输出格式

```
标题：[用户提供的标题]

一、概述
[内容...]

二、存在问题
（一）[问题一标题]
[内容...]

（二）[问题二标题]
[内容...]

三、建议
（一）[建议一标题]
[内容...]

（二）[建议二标题]
[内容...]

（三）[建议三标题]
[内容...]
```

## 写作原则

1. **立场正确**：符合党和国家方针政策
2. **实事求是**：数据真实，问题准确，建议可行
3. **简明扼要**：语言简练，重点突出
4. **结构清晰**：层次分明，逻辑严密
5. **建议为先**：落脚点是提出有价值的建议

## 注意事项

- 社情民意不是学术论文，不需要过多理论阐述
- 社情民意不是投诉信，语气要客观理性
- 社情民意要有建设性，重在提出解决方案
- 涉及敏感话题时要把握尺度，聚焦政策建议而非批评指责
- 作为九三学社社员，应体现参政党成员的担当，关注科技、教育、医卫、生态等领域
